### Installation

```
git clone https://github.com/openopen114/iuapCommander
cd iuapCommander
npm install
npm link
```



### Commands

` g <type> <name> `   generate page (p) / component(c)



#### generate  page

`iuap g p MY-PAGE-NAME`



#### generate component 

`cd MY-PAGE-NAME`

`iuap g c my-table`
